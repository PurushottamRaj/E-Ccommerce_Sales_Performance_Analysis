#E-Commerce Sales Analysis Dashboard (Power BI)

#**Project Overview**

This project analyzes an E-Commerce Sales Dataset using Microsoft Power BI to extract meaningful business insights. The goal is to transform raw sales data into an interactive dashboard that helps businesses understand their performance, identify trends, and make data-driven decisions.

The dashboard provides insights into sales performance, customer purchasing behavior, product category performance, regional sales distribution, and payment mode usage.

#**Problem Statement**

E-commerce platforms generate large amounts of transactional data including orders, customers, products, and payments. However, businesses often struggle to convert this raw data into actionable insights.

Key challenges include:
-Identifying top-performing products and categories
-Understanding regional sales distribution
-Analyzing customer purchasing behavior
-Tracking profit trends over time
-Understanding preferred payment modes

Without proper visualization and analysis tools, decision-makers cannot effectively optimize business strategies.

#**Proposed Solution**

To solve this problem, a Power BI dashboard was developed that:
-Cleans and transforms raw data using Power Query
-Creates calculated metrics using DAX
-Builds interactive visualizations to track business performance

The dashboard provides insights such as:
-Total Sales
-Total Quantity Sold
-Total Profit
-Average Order Value (AOV)
-Sales by State
-Profit by Month
-Sales by Category
-Payment Mode Distribution

These insights help businesses identify opportunities, improve profitability, and optimize operations.

#**Tools & Technologies Used**

-Power Query (Data Cleaning & Transformation)

#**Key Insights**

Some important insights derived from the dashboard include:
-Clothing category contributes the highest sales volume
-Maharashtra and Madhya Pradesh generate the highest sales
-Cash on Delivery (COD) is the most used payment method
-Some months show lower or negative profit
-Printers and Bookcases are among the most profitable sub-categories
